# Фото для сайта

Положите сюда свои изображения — после этого напишите в чат, и они будут подключены на сайте.

## Куда класть

```
public/images/
├── products/     ← фото товаров (торты, пирожные и т.д.)
├── categories/   ← фото для 5 категорий (необязательно)
└── about/        ← фото для страницы «О нас» (необязательно)
```

## Формат

- **JPG** или **WebP** — лучше всего
- **PNG** — тоже подойдёт
- Размер: **800×600 px** или больше (сожмём при необходимости)
- Вес: до **500 KB** на фото — сайт будет грузиться быстрее
